## Mobile SDK功能 

定制专属移动 APP，为无人机用户提供你想要的一切 ！

通过 Autel Mobile SDK，开发者可以实现自主飞行，控制相机和云台，接收实时视频图传和传感器数据，下载媒体文件，以及实现多种智能任务等等。

[Autel开发者平台](https://develop.autelrobotics.cn) 为开发者提供Mobile SDK软件开发工具包，入门文档，API说明文档，教程文档，沟通平台等功能；
